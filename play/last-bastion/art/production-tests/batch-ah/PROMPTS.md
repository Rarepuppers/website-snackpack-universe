# Batch AH prompt provenance

Mode: four built-in image-generation calls. Boundary and decal sheets used magenta chroma; the multi-biome fixture sheet used yellow chroma. All were converted with the installed image-generation helper using border sampling, soft matte, and despill.

- **Floor:** exact 4×4 opaque frontier sheet partitioned by row into Earth, hostile planet, war-torn surface, and additional frontier variants.
- **Boundary:** exact 4×2 magenta-backed ridge family with cardinal sections, corners, bunker gate, and meteor breach.
- **Fixtures:** exact 4×2 yellow-backed family: dead tree, cave boulder, ruined shelter, alien crystal, frozen boulder, marsh fungus, meteor, obelisk.
- **Decals:** exact 4×2 sparse magenta-backed underlays: ruts, leaves, gravel, ash, snow, marsh residue, rubble, impact scorch.

Shared prompt templates and negative constraints: `../../../environment-prompts.md`.
