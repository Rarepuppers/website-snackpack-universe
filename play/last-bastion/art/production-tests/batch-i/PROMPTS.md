# Batch I prompt contract

Built-in image generation used the approved F1 action-tile atlas as the strict style reference. Weapon tiles requested eight signature silhouettes at 96px; perk tiles requested eight authored upgrade motifs at 64px; hotkey tiles requested eight action/empty/disabled motifs at 64px. Every prompt required flat `#FF00FF` chroma, no text, no numbers, no bindings, no baked cooldown shadows, and readability beneath code-rendered overlays.

## I2 — slot/tier UI master

Built-in generation used the promoted weapon atlas and F1 action atlas as strict material references. The prompt requested a regular 4 × 4 sheet: five silhouette-distinct class frames, stash, Tier I–III borders, three discard states, merge base/pulse, drag ghost, and neutral utility frame. Flat `#FF00FF`; no text, weapon art, bindings, or baked legality/selection.

## I3 — placement/shop UI master

Built-in generation used the N2 terminal panel and promoted weapon atlas as strict references. The prompt requested a separated 2 × 2 source sheet containing an empty placement modal, empty stat card, empty salvage-counter backdrop, and sell/buy/merge glyph strip. Flat `#FF00FF`; no text, numbers, prices, weapons, bindings, or baked state.
