# Batch H prompt contract

Mode: built-in image generation with Batch A floor, boundary, and obstacle atlases as strict layout/collision references. Each master requested flat `#FF00FF` chroma, exact runtime grids, unchanged footprints, premium chunky pixel clusters, and no text/UI/characters/projectiles.

- Emberfall floor: 3 × 2, clean/alternate/seam/hatch/cracked/ember-edge frames.
- Emberfall boundary: 4 × 2, straight/vertical/corners/junctions/damaged/gate pieces.
- Emberfall obstacles: 2 × 2, barricade/crate/conduit/ember-slag re-dress with preserved footprints.
- Emberfall decals: 3 × 2, low-contrast soot/crack/glow/weld/ash/molten-seam underlays.

Generated chroma sources are retained unchanged; local extraction and nearest-neighbour normalization are deterministic and repeatable.
