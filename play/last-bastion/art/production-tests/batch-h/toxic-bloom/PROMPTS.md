# Toxic Bloom variant prompt contract

Mode: built-in image generation with the Emberfall Batch H atlases as strict layout, collision, and pixel-density references. The four masters requested flat `#FF00FF` chroma, unchanged runtime grids and footprints, dark teal alloy, muted violet growth, restrained lime bioluminescence, and no text/UI/characters/projectiles.

- Floor: 3 × 2 clean/alternate/seam/hatch/cracked/biomass-edge frames.
- Boundary: 4 × 2 straight/vertical/corners/junctions/damaged/biomass-gate pieces.
- Obstacles: 2 × 2 vine barricade, growth-sealed crate, moss conduit, purple/lime biomass obstruction.
- Decals: 3 × 2 mold/cracks/glow/spores/moss/vine underlays with intentionally subdued contrast.
