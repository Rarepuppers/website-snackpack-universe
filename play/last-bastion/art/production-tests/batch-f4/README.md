# Production Asset Batch F4 — Grenade Tube

The original Bastion Grenade Tube family includes four 96-pixel ring states and an eight-frame 64-pixel projectile/effect atlas. Frames are idle, armed, fire, and reload; effects are grenade projectile, fuse trail, bounce, warning pulse, explosion core, blast ring, cover impact, and ready glint.

Projectile flight, collision, explosion radius, damage, knockback, and warning timing remain code-driven. Chroma sources and full-resolution transparent masters are retained for higher-resolution Steam rendering.
