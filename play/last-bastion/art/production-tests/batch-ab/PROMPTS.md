# Batch AB prompt provenance

Built-in image generation produced four independent flat-magenta masters.

- Body: exact 4x11 Storm Regent directional/state grid; tall insulated command chassis, asymmetric conductor crown, one grounded cable, compact chest coil; no gameplay geometry.
- Node: exact 6x1 conductive-node lifecycle with a stable low footprint, broken critical insulator, and collapsed non-live wreck.
- Effects: exact 4x2 onset/dissipate atlas for chain origin, node overcharge, coil discharge, and node interruption.
- Portrait: square central-safe-area command bust preserving crown, cable, and chest coil at small scale.

Every prompt required a uniform `#ff00ff` chroma background and prohibited separate nodes in the body, chain rails/hops, endpoints, cover stops, rings, timing ticks, health bars, damage widths, text, UI, shadows, watermarks, and logos. Chroma was removed with the installed imagegen helper using border auto-key, soft matte, and despill.
