# Foundry Fabricator production package — Batch Y

- Fabricator: 4×9 directional/state sheet, 36 frames at 192×192.
- Pad: six stable progress/outcome states at 128×128.
- Drone: 4×7 directional/state sheet, 28 frames at 128×128.
- Turret: 4×8 directional/state sheet, 32 frames at 128×128.
- Effects: four local event families × onset/dissipate, eight frames at 128×128.

Untouched chroma sources, clean-alpha masters, deterministic nearest-neighbour derivatives, prompts, adaptive-row normalization, and contact-sheet QA are retained here. Reservations, pad health/progress, child ownership/lifetimes/caps, turret targeting/lane, cover blocking, collision, damage, and timing remain code-owned.
