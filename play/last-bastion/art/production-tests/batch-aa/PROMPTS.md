# Batch AA prompt provenance

Built-in image generation produced four independent flat-magenta masters.

- Body: exact 4x11 Assembly Prime directional/state grid; broad industrial command chassis, asymmetric forge shoulder, three emitter vanes, articulated recall arm; no gameplay geometry.
- Pad: exact 6x1 reinforced command-pad lifecycle with a stable low footprint and silhouette-readable progress/damage.
- Effects: exact 4x2 onset/dissipate atlas for emitter lock origin, fabrication completion, recall lock/snap, and shutdown.
- Portrait: square central-safe-area command bust preserving forge shoulder, three vanes, and recall arm at small scale.

Every prompt required a uniform `#ff00ff` chroma background and prohibited pads/children in the body, lane rails, timing beads, tethers, target rings, damage areas, text, health bars, UI, shadows, watermarks, and logos. Chroma was removed with the installed imagegen helper using soft matte and despill.
