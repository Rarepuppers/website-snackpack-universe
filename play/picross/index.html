How to play Picross (Nonograms)

Picross — also called Nonograms, Griddlers or Hanjie — is a picture logic puzzle. Each row and column has a list of numbers; each number is the length of a run of filled cells, given in order, with at least one empty cell between runs. By cross-referencing the row and column clues you can deduce exactly which cells must be filled, and a hidden picture gradually appears. There's no guessing in a well-made nonogram — every cell can be reasoned out.

Tips

  • Start with the big numbers. A clue close to the row or column length overlaps itself — those overlap cells are always filled.
  • Cross out the empties. Marking cells you know are blank is just as important as filling — it stops you re-checking them and reveals new deductions.
  • Work the edges. A clue starting at an edge fixes the first run's position immediately.
  • Switch between rows and columns. Each fill you place gives the crossing line new information.

Grid sizes

  • 5×5 — quick, friendly puzzles to learn the rules.
  • 10×10 — the classic size, a satisfying few minutes each.
  • 15×15 — larger pictures with more deduction.
  • Daily — one shared puzzle that changes every day.
