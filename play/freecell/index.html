How to play FreeCell

Move all 52 cards to the four foundations, building each suit from Ace through King. Unlike Klondike, every card starts face up, so the puzzle is about planning. Build cascades downward in alternating colours, use the four free cells as temporary parking spaces, and preserve empty columns whenever you can.
