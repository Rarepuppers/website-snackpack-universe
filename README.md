This folder is an extracted copy of the new SnackPack site.

Recommended final project path:

- `C:\godot\website-snackpack`

When you move it into its own folder or repo, keep these files at the project root:

- `index.html`
- `site.css`
- `sitemap.xml`
- `robots.txt`
- `apps/`
- `privacy/`
- the `snackpack-*` image and icon assets

The pages were built to work as the root of a standalone GitHub Pages site.
